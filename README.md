==
Presentation
==

Create presentations from Markup


Installation :
- Step1: Install compound -

```
       sudo npm install '1602/compound' -g
```

- Step2: Clone the repository
- Step3: run command -

```
       npm install -l
```

- Step4: Run the app server 

```
       compound s 8888
```

- Step5: Your app is up and running on http://localhost:8888
